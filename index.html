I'm a passionate web developer

Saya adalah mahasiswa S1 Program Studi Pendidikan Guru Sekolah Dasar (PGSD) di Universitas Pendidikan Indonesia Kampus Sumedang, angkatan 2024. Saat ini saya sedang menempuh proses belajar untuk menjadi calon pendidik yang tidak hanya memahami materi, tetapi juga mampu menyampaikan pembelajaran dengan cara yang menarik dan mudah dipahami.

Portofolio ini saya susun sebagai bagian dari pemenuhan tugas pada mata kuliah Literasi TIK dan Media Pembelajaran. Melalui portofolio ini, saya mencoba mengumpulkan dan menampilkan hasil belajar, pemahaman, serta pengalaman saya dalam memanfaatkan teknologi sebagai media pembelajaran.
